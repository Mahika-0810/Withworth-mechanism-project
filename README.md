Whitworth Quick Return Mechanism Simulator

An interactive desktop application that simulates the Whitworth Quick Return Mechanism with real-time animation, complete kinematic analysis, and dynamic force analysis. The simulator is built using Python, Tkinter, Matplotlib, and NumPy to help students and engineers visualize the working principles of quick return mechanisms.

Features
🎥 Real-time Whitworth mechanism animation
⚙️ Adjustable mechanism parameters
▶️ Start, Pause and Reset controls
🎚 Adjustable animation speed
🔍 Interactive Zoom & Pan
📊 Position, Velocity and Acceleration diagrams
📈 Dynamic torque analysis using the Virtual Power Method
📌 Supports multiple analysis positions (A, B and C)
🥧 Animated rotating object attached to the slider
📐 Automatic mechanism synthesis from user inputs
🖥 Fullscreen visualization mode
🎨 Clean and modern graphical interface
Technologies Used
Python 3.x
Tkinter
NumPy
Matplotlib
